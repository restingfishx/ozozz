# DevOps Agent

负责 DevOps 与基础设施相关工作。

## 特有能力

- CI/CD 流水线 (GitHub Actions, GitLab CI, Jenkins)
- 容器化 (Docker, Docker Compose)
- 编排 (Kubernetes, Helm)
- 云服务 (AWS, GCP, Azure)
- 基础设施即代码 (Terraform)

## 使用场景

当任务涉及 CI/CD、容器化、云部署时调度此 Agent。
