> 高端跨境电商独立站

---

## 1. 设计风格

### 1.1 整体定位

| 属性 | 值 |
|------|-----|
| 风格 | 极简高端、大量留白、大牌质感 |
| 参考 | Ouraring、Therabody、Aesop |
| 品牌感 | 科技健康、简约高级 |
| 目标人群 | Z 世代 |

### 1.2 关键词

- 极简 · 高端 · 留白
- 柔和 · 细腻 · 质感
- 流畅 · 快速 · 转化

## 3. 字体系统

### 3.1 字体选择

| 用途 | 字体 | 字重 |
|------|------|------|
| 标题 | Inter | 600 / 700 |
| 正文 | Inter | 400 / 500 |
| 价格 | Inter | 600 |
| 辅助 | Inter | 400 |

### 3.2 字号层级

| 元素 | 桌面 | 移动端 | 行高 |
|------|------|--------|------|
| h1 (Hero标题) | 48px | 32px | 1.2 |
| h2 (区块标题) | 32px | 24px | 1.3 |
| h3 (卡片标题) | 20px | 18px | 1.4 |
| body (正文) | 16px | 14px | 1.6 |
| small (辅助) | 14px | 12px | 1.5 |
| caption (标签) | 12px | 11px | 1.4 |

### 3.3 字重使用场景

- 400 (Regular): 正文、描述
- 500 (Medium): 按钮文字、次要标题
- 600 (Semibold): 价格、重要标题
- 700 (Bold): 不常用，避免过度加粗

## 8. 组件设计

### 8.1 按钮 (Button)

| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 填充 #1A1A1A，白色文字 | 主要操作（加购、结算） |
| Secondary | 边框 #1A1A1A，透明背景 | 次要操作 |
| Ghost | 无边框，透明背景 | 辅助操作 |
| Danger | 填充 #EF4444 | 删除、取消 |

**尺寸**:
- sm: 高度 32px，用于小型按钮
- md: 高度 40px，标准按钮
- lg: 高度 48px，用于主要行动点

**状态**:
- Default: 正常状态
- Hover: 背景加深 10%
- Active: 缩放 scale(0.98)
- Disabled: 透明度 50%，不可点击
- Loading: 显示 spinner

### 8.2 输入框 (Input)

- 高度: 44px
- 边框: 1px solid #E5E7EB
- 圆角: 8px
- 聚焦: 边框 #1A1A1A，2px

### 8.3 卡片 (Card)

- 背景: #FFFFFF
- 边框: 1px solid #F3F4F6
- 圆角: 12px
- 阴影: hover 时显示 md
- 内边距: 24px

### 8.4 商品卡片

```
┌─────────────────────────┐
│  ┌─────────────────┐   │
│  │                 │   │  ← 图片区域 (1:1 或 4:3)
│  │    Product      │   │
│  │    Image        │   │
│  │                 │   │
│  └─────────────────┘   │
│                         │
│  Category Tag           │  ← 可选：分类标签
│  Product Name           │  ← 标题，2行 max
│  $29.99                 │  ← 价格
└─────────────────────────┘
```

### 8.5 导航栏

- 高度: 64px (桌面) / 56px (移动)
- 背景: 白色，固定顶部
- 阴影: sm (滚动后显示)
- Logo: 左侧
- 菜单: 右侧（桌面）/ 汉堡菜单（移动）

### 8.6 底部导航 (移动端)

- 不需要，页面跳转少，用顶部导航即可

## 10. 动效设计

### 10.1 过渡时间

| 场景 | 时长 | 缓动 |
|------|------|------|
| 按钮悬停 | 150ms | ease |
| 页面切换 | 200ms | ease-out |
| 弹框显示 | 250ms | cubic-bezier(0.16, 1, 0.3, 1) |
| 列表加载 | stagger 50ms | ease |

### 10.2 反馈动画

- 按钮点击: scale(0.98)
- 卡片悬停: translateY(-4px) + 阴影增强
- 加载: 骨架屏 (skeleton)
- 成功: 绿色 toast 淡入
- 错误: 红色边框 + 错误信息淡入

### 10.3 滚动行为

- 平滑滚动: scroll-behavior: smooth
- 滚动显示 Header 阴影

## 11. 可访问性

- 对比度: 文字与背景至少 4.5:1
- 焦点状态: 可见的 focus ring
- 按钮尺寸: 最小 44px (移动端)
- 图片 Alt: 所有商品图片必须有 alt

---

## 12. 设计资源

### 12.1 图标

使用 Heroicons 或 Lucide：
- 线条风格 (outline)
- 24px 标准尺寸

### 12.2 图片

- 商品图片: 800x800px 以上
- Banner: 1920x600px (桌面) / 800x600px (移动)
- 格式: WebP 优先
