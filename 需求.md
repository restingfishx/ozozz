实现一套可直接运营的高端跨境电商独立站。
产品：眼罩、口罩、个人护理、日用防护用品。

## 技术栈
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Supabase PostgreSQL
- Stripe Checkout
- 英文站
- 移动端优先响应式

## 设计风格（高端纯电商）
参考：ouraring.com、therabody.com
要求：
- 极简高端、大量留白、大牌质感
- 排版清晰、体验流畅、购物路径短
- 柔和圆角、细腻阴影、高级不浮夸
- 低饱和配色：黑白灰 + 深蓝/墨绿/深咖
- 视觉专业、国际范、高转化、不卖廉价感

## 【C端 - 标准购物商城（必须完整电商功能）】
1. 首页
   - 轮播 Banner
   - 商品分类
   - 热销/新品商品模块
2. 商品列表页
   - 筛选、排序、搜索、分页
3. 商品详情页
   - 多规格（颜色/尺寸）
   - 加购物车
4. 购物车
   - 修改数量、删除、合计金额
5. 结算页
   - 收货地址表单
   - 提交订单
6. 支付
   - 跳转到 Stripe Checkout
   - 支付成功/失败页
7. 我的订单
   - 查看订单、支付状态、物流状态
8. 政策页
   - 关于我们、退换货政策、隐私政策、服务条款

## 【B端 - 管理后台】
1. 管理员登录
2. 商品管理：增删改查、上下架
3. 订单管理：查看、修改状态、填写物流单号
4. 简单数据统计

## 数据库（提供 Supabase SQL）
- products 商品表
- orders 订单表
- order_items 订单商品表
- addresses 收货地址表
- admin 管理员表

## API 接口
- 商品接口
- 购物车接口
- 创建订单、支付、回调
- 订单查询接口
- 后台管理接口
- 管理员登录接口

## 输出要求
1. 完整项目结构
2. 所有页面代码（C端 + B端）
3. 所有 API 逻辑
4. 可直接执行的建表 SQL
5. 环境变量配置
6. Vercel 部署步骤
7. 代码可直接运行、无缺失、无报错
