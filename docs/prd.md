# PRD - Ozozz 高端跨境电商独立站

> v1.0 | 2025-03-17

---

## 1. 产品概述

### 1.1 产品定位

- **产品名称**：Ozozz 独立站
- **产品类型**：中高端跨境电商独立站
- **核心价值**：为 Z 世代消费者提供设计感强、质感高端的助眠类日用品，主打功能对标竞品，但通过设计差异化建立品牌认知

### 1.2 背景与动机

- 创始人（你）是跨境电商小白，希望通过独立站建立品牌
- 目标：前期通过高颜值、高逼格的设计吸引 Z 世代流量，后期拓展品类覆盖全人群
- 多渠道布局：独立站 + 亚马逊 + TikTok Shop 同步运营

### 1.3 目标用户

| 维度 | 描述 |
|------|------|
| 年龄 | Z 世代（18-30 岁），后期扩展至全年龄 |
| 消费特征 | 注重设计和颜值，愿意为"好看"付费，追求生活仪式感 |
| 触达渠道 | 小红书、Instagram、TikTok 等社交媒体种草 |
| 价格敏感度 | 中端价格敏感度，可接受 $20-40 价位 |

## 2. 市场分析

### 2.1 竞品参考

| 品牌 | 价格 | 特点 |
|------|------|------|
| Ouraring | $99 | 智能眼罩，科技+睡眠概念，极简高逼格 |
| Manta Sleep | $35-45 | 100%遮光，硅胶材质，包装简约 |
| Alaska Bear | $25-35 | 丝绸材质，亚马逊爆款 |
| Lilmed | $20-30 | 法国品牌，设计师款 |

### 2.2 差异化机会

- 现有品牌要么偏科技风，要么偏基础款
- **设计感 + 生活方式** 定位有空档
- 可参考 Warby Parker 玩法——"不只是功能品，也是配饰"

## 3. 功能需求

### 3.1 优先级分类

| 优先级 | 功能 | 说明 |
|--------|------|------|
| **P0** | 首页、轮播 Banner、商品分类、热销/新品模块 | 流量入口 |
| **P0** | 商品列表页（筛选/排序/搜索/分页） | 选购入口 |
| **P0** | 商品详情页（多规格、加购物车） | 转化核心 |
| **P0** | 购物车（数量修改、删除、合计） | 结算前置 |
| **P0** | 结算页（收货地址、提交订单） | 必做 |
| **P0** | Stripe 支付（Checkout、回调、结果页） | 交易闭环 |
| **P0** | 我的订单（查看订单、支付状态、物流状态） | 用户必做 |
| **P1** | 管理员登录 | B 端入口 |
| **P1** | 商品管理（增删改查、上下架） | B 端核心 |
| **P1** | 订单管理（查看、修改状态、填写物流单号） | B 端核心 |
| **P1** | 数据统计 | B 端辅助 |
| **P2** | 政策页（关于我们、退换货政策、隐私政策、服务条款） | 信任建设 |
| **P2** | FAQ 页面 | 客服辅助 |

### 3.2 C 端功能详情

#### 首页
- 轮播 Banner（3-5 张，营销活动/新品推广）
- 商品分类入口
- 热销商品模块
- 新品商品模块
- 底部：政策链接、社交媒体链接

#### 商品列表页
- 筛选：按类别、价格区间
- 排序：销量、 price (低到高/高到低)、最新
- 搜索：关键词搜索
- 分页：每页 12-24 个商品

#### 商品详情页
- 商品图片（多图展示）
- 标题、价格
- 规格选择（颜色、尺寸）
- 加购物车按钮
- 商品描述、材质说明

#### 购物车
- 商品列表（图片、标题、规格、数量、单价）
- 修改数量
- 删除商品
- 合计金额
- 去结算按钮

#### 结算页
- 收货地址表单（姓名、电话、地址、国家、城市、邮编）
- 订单摘要
- 提交订单按钮

#### 支付
- Stripe Checkout 跳转
- 支付成功页（跳转回站内）
- 支付失败页（跳转回站内）

#### 我的订单
- 订单列表
- 订单详情（商品、金额、地址、物流单号）
- 支付状态显示

#### 政策页
- 关于我们
- 退换货政策
- 隐私政策
- 服务条款

### 3.3 B 端功能详情

#### 管理员登录
- 用户名 + 密码登录
- Session 管理

#### 商品管理
- 商品列表（展示上下架状态）
- 新增商品（名称、价格、描述、图片、规格、库存）
- 编辑商品
- 删除商品
- 上架/下架

#### 订单管理
- 订单列表
- 查看订单详情
- 修改订单状态（待支付/已支付/已发货/已完成/已取消）
- 填写物流单号

#### 数据统计
- 总订单数
- 总销售额
- 待处理订单数

## 5. 运营方案

### 5.1 物流方案

| 阶段 | 方案 |
|------|------|
| 前期（代发） | 国内供应商一件代发，包装自己控制，发货地标注为中国，物流选可追踪（4PX/云途） |
| 后期（有量） | 囤到海外仓（美国/欧洲），缩短物流时效 |

### 5.2 支付方案

- 货币：前期只做美元，网站可显示多币种（EUR/GBP/AUD）
- 支付方式：Stripe（信用卡、Apple Pay、Google Pay）
- 后期可加 Klarna（先买后付）

### 5.3 客服方案

| 阶段 | 方案 |
|------|------|
| 前期 | 邮件 + 网站联系表单 + FAQ 页面 |
| 后期 | 加 Tawk.to 或 Intercom（即时聊天） |

## 6. 验收标准

### 6.1 功能验收

- [ ] C 端所有页面可正常访问
- [ ] 完整购物流程可走通（浏览→加购→结算→支付→订单）
- [ ] 支付成功后订单状态正确更新
- [ ] B 端管理员可正常登录
- [ ] B 端商品管理功能完整
- [ ] B 端订单管理功能完整

### 6.2 视觉验收

- [ ] 设计风格符合"极简高端、大量留白、大牌质感"
- [ ] 柔和圆角、细腻阴影
- [ ] 低饱和配色（黑白灰 + 深色点缀）
- [ ] 移动端响应式正常

### 6.3 性能验收

- [ ] 首页加载时间 < 3s
- [ ] 支付流程顺畅，无卡顿
